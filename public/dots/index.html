Dotfiles

Here is a small overview of the most important parts of my dots. Check out the repo for all the configurations.

Operating System: NixOS

Window Manager: DWM

DWM is the (arguably) final stage when going down the hyper-minimal linux road.

Previously I have used Hyprland which is an amazing an customizable window manager. If you like the DWM look but want flexible customization and lot’s of plugins, you might want to check out i3.

Terminal: ST

Simple Terminal is a very minimal terminal and does just enough for me. If you want some extra features or simple additions you can check out the patches. The only patches I use:

  • Anysize
  • Ligeratures

Simple Terminal is created to be minimal. If you want more features, check out these great alternatives:

  • Wezterm
  • Kitty
  • Alacritty

Editor: Helix

A terminal text editor for the cool kids. Helix has a modern approach on the vim keybinding with plenty of built-in features and is programmed in rust 👀.

Browser: Firefox

Some CLI magic

  • Zoxide is a smarter cd command, inspired by z and autojump.
  • Eza, a modern, maintained replacement for ls.
  • Starship
  • Zathura
